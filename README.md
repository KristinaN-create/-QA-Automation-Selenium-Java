# QA Automation Project - SauceDemo

This project contains automated UI tests for the [SauceDemo](https://www.saucedemo.com/) website. It was developed as a final project for the Software QA Automation course.

## 🚀 Scenarios Covered
* *Login Functionality*: Validating successful login and handling locked-out users.
* *Shopping Cart*: Adding and removing items from the cart.
* *Checkout Process*: Complete end-to-end flow from cart to order confirmation.

## 🛠️ Tech Stack
* *Language*: Java
* *Automation Library*: Selenium WebDriver
* *Test Framework*: TestNG
* *IDE*: IntelliJ IDEA
